# ✨ لكجري نيش | Luxury Niche — محوّل الهوية التجارية

تطبيق Streamlit متكامل لتحويل بيانات متجر "مهووس" إلى هوية "لكجري نيش" الفاخرة.

---

## 🚀 التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📋 المميزات

### 1. تبويب تحويل المنتجات (بالـ AI)
- يقبل ملف XLSX بتنسيق Salla الشامل
- يستخرج المكونات الحقيقية من HTML باستخدام BeautifulSoup
- يرسل لـ Claude Opus وصفاً نيش فاخراً بـ HTML احترافي
- يحفظ النتائج في Session State (لا فقدان عند التحديث)
- شريط تقدم + سجل مباشر لكل منتج

### 2. تبويب تحويل الماركات (بالـ AI)
- يقبل ملف CSV بتنسيق ماركات Salla
- يحوّل الوصف، عنوان الصفحة، رابط SEO، ووصف Meta
- يعيد البيانات بتنسيق JSON منظم

### 3. تبويب التطهير الفوري (بدون AI)
- معالجة فورية لكل 7,604 منتج و521 ماركة دفعة واحدة
- تقرير تفصيلي بعدد الخلايا المُطهَّرة في كل عمود
- تطهير نص مباشر في الواجهة

## 🔑 إعداد API

أدخل مفاتيح Anthropic في الشريط الجانبي (مفتاح في كل سطر):
```

sk-ant-api03-yyyyy
```

يتم التدوير بين المفاتيح تلقائياً للتسريع ومعالجة حدود Rate Limit.
